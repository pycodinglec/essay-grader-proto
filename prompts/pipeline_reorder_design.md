# 파이프라인 순서 변경 설계

- 날짜: 2026-02-23

## 배경

기존에는 에세이 업로드가 먼저, 채점기준표 업로드가 나중이었으나, 채점기준표가 평가의 전제 조건이므로 먼저 업로드받도록 변경한다.

## 변경 사항

1. 채점기준표 업로드를 Step 1로 이동 (에세이 업로드 전)
2. 에세이 업로드는 채점기준표 검증 후에만 표시 (Step 2)
3. OCR 수동 버튼 제거 → 에세이 파일 처리 버튼 클릭 시 자동 OCR 실행
4. OCR에 진행률 바 추가 (기존 spinner → progress bar + status text)
5. 제출물 식별 결과는 Step 3으로, 채점은 Step 4로, 다운로드는 Step 5로 번호 변경
